# Life Tracker 🔥

Persönlicher Life Tracker mit Gewohnheiten, Tageszielen, Health-Tracking und Dashboard.

## Features
- **Habits** – Gewohnheiten tracken mit Streaks, Icons und Farben
- **Tagesziele** – Tägliche Ziele mit Prioritäten (Hoch/Mittel/Normal)
- **Health** – Wasser, Schlaf, Schritte, Gewicht, Stimmung, Energie
- **Dashboard** – Wochenüberblick, 30-Tage Heatmap, Streak Ranking, Badges
- **Notizen** – Tägliche Reflexionen und Gedanken
- **9 Badges** – Achievements die automatisch freigeschaltet werden
- **PWA** – Installierbar auf dem Handy als eigenständige App

## Schnellstart (CodeSandbox)
1. Geh auf [codesandbox.io](https://codesandbox.io)
2. Erstelle ein neues React-Projekt
3. Ersetze `src/App.js` mit der Datei aus diesem Projekt
4. Fertig!

## Deploy auf Vercel (empfohlen)
1. Push dieses Projekt auf GitHub
2. Geh auf [vercel.com](https://vercel.com) und logge dich mit GitHub ein
3. Klicke "Import Project" → wähle dein Repo
4. Vercel erkennt React automatisch → "Deploy"
5. In 60 Sekunden hast du eine URL wie `life-tracker-xyz.vercel.app`

## Auf dem Handy installieren
1. Öffne deine Vercel-URL im Browser (Chrome/Safari)
2. **Android**: Tippe auf ⋮ Menü → "Zum Startbildschirm hinzufügen"
3. **iPhone**: Tippe auf Teilen-Button → "Zum Home-Bildschirm"
4. Die App öffnet sich jetzt wie eine native App ohne Browser-UI

## Daten
Alle Daten werden lokal im Browser gespeichert (localStorage).
Kein Server, kein Account, 100% privat.
